IntelliAI 插件架构图

IntelliJ IDEA 插件 ↔ Spring Boot API ↔ 家庭 LLM 集群（WireGuard 覆盖网络）

客户端与边缘

  • • IntelliJ IDEA + intelli-ai-engine（统一 AI SPI）
  • • 8 个业务插件：javadoc / changelog / tracer / nacos / swagger / repairer / terminal / dev-helper
  • • React 19 + Vite 7 SPA，Hash 路由，i18n（中 / 英）
  • • 阿里云 ECS 上的 Nginx：TLS 1.2/1.3、限流、SSE 透传
  • • HMAC-SHA256 请求签名 + X-Device-Id + User-Agent 白名单

应用与算力

  • • zeka-stack-api — Spring Boot 3 · Java 17 · 端口 8080 · /api
  • • 模块：auth · feedback · model · statistics · project · freeai · monitor · manager/github
  • • new-api（Docker）— OpenAI 兼容调度器，部署于 Mac mini M4 :3000
  • • Mac Studio M2 Ultra / Mac mini M4 / Mac mini M2 — Ollama + LM Studio
  • • WireGuard 覆盖网络打通 ECS ↔ 家庭网络（192.168.31.0/24）

集成与数据

  • • GitHub — OAuth 应用 · GraphQL Discussions · REST Issues · Webhook
  • • MySQL 8（zeka-stack-helper）部署于 M4 — 认证 / 会话 / 事件
  • • Zhipu AI — 模型列表兜底 + 代理推理
  • • 25+ 直连 AI 服务商，由插件使用用户密钥直接调用
  • • Mac mini M2 上的 HTTP 出口代理 :6152 用于访问海外 SaaS
